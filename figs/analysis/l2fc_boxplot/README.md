# Log2FoldChange Boxplots

Folder contains boxplots of log2FoldChange gene counts and pathway GSVA scores colored by diagnosis.
The statistical test employed for log2FoldChange gene counts boxplots is the Wilcoxon rank sum test and
for log2FoldChange GSVA scores boxplots it is the Kruskal-Wallis test. The GSVA scores were calculated directly from
a subject-wise log2FoldChange matrix.
